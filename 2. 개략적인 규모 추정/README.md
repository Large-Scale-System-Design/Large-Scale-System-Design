# 개략적인 규모 추정

## 개요

이 장에서는 시스템 설계 시 필요한 자원과 규모를 추정하는 방법론을 소개합니다. 실제 서비스의 트래픽, 저장 공간, 대역폭 등의 요구사항을 정확하게 추정하는 것은 효율적인 시스템 설계의 기초가 됩니다.

## 주요 내용

- **용량 추정**: QPS(Queries Per Second), 읽기/쓰기 비율 등 트래픽 추정
- **저장 용량 계산**: 데이터 크기, 복제본 수, 증가율을 고려한 저장 공간 계산
- **대역폭 추정**: 네트워크 트래픽과 데이터 전송량 계산
- **메모리 사용량 추정**: 캐시 크기와 메모리 요구사항 계산
- **서버 수 추정**: 필요한 서버 대수와 자원 배분 계획
- **단위 변환**: 바이트, 비트, 초, 분 등의 단위 변환과 계산
- **근사치 활용**: 복잡한 계산을 단순화하는 근사치 활용 방법

## 학습 목표

- 시스템 요구사항을 정량적으로 추정할 수 있다
- 용량, 저장 공간, 대역폭 등을 정확하게 계산할 수 있다
- 근사치를 활용하여 빠르고 정확한 규모 추정을 수행할 수 있다
- 면접에서 규모 추정 문제를 효과적으로 해결할 수 있다
